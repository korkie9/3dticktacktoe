This version of tick tack toe can be conceptualised as a 3D redition of the original game represented on a 2D plane.

Each 3x3 block square can be considered a layer of a cube.

Just like in regular tick tack toe, getting three symbols in a row will win you the game, be it vertically, horizonally, diagonally across either plane.

Visually, this means for either plane if one player gets 3 in a row on one layer, just like in regular tick tack toe, that player wins.

Also if they get 3 in a row across the cube, they win, be it from one corner to the other ro from the "top" to the "bottom"
